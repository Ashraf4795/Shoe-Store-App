# Shoe Store app

Shoe store android application to add new shoe items, this project is part of Udacity adavanced android developmen with kotlin.

### Dependencies

```
- Lottie
- Coroutine
- ConstraintLayout
- Navigation component
- Gson
- Lifecycle
```
